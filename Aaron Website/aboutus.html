Project Information

I created this website with a mix of HTML, CSS, Javascript files and a separate folder of external assets for things like images and svg icons. The home page is defined by the HTML code in an index.html file while each of the other pages also has their specific .html file that closely matches the name on the website (about.html, car_list.html,etc). Three of the files have corresponding Javascript files to configure special content such as the analytics charts (analytics.js) or handle form validation and data processing (home.js and car_list.js). There is a shared CSS file named main.css which is linked to all of the HTML pages and here I make changes to customize the style of specific content across all pages. For example, in the main.css I set all page titles to a royal blue color by customizing the h1 tag or set the source caption for the analytics graphs to a light gray using a CSS class ( p.source).

This site heavily utilizes the Bootstrap framework for both a professional styling and easy, out-of-the-box, interactive functionality. I include both the CSS and Javascript from Bootstrap via the CDN which is linked to each page in a script tag. Each page on the site has a consistent, branded navigation bar at the top and footer at the bottom that comes right from Bootstrap's example snippets. For the charts on the Analytics page, I've utilized the Chart.js framework and read their documentation to add my own data and customize style elements such as line colors, data labels, and axis settings. I pulled data for the charts from two external sources that I've cited right on the page and the home page has links to a third-party site (greencars.com) which provide nice explanations of how different types of EVs operate. The other content on the home page I generated with ChatGPT and pasted it into the HTML file. All source files for this site can be accessed from my Github link shown below.
